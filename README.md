# Simple clock face - Simple pale

Simple experimental clock face for fitbit OS.
## Includes:

❤️ heart rate, 

👣 daily step count,

🕑 current date and time.

Supports both 12h/24h format.
Only english supported locale.
Works in Sense and Versa 3.
